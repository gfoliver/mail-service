# Mail Service

A NodeJS express mail sending service

## Built With

* [NodeJS](https://nodejs.org/)
* [Express](https://expressjs.com/)
* [Nodemailer](https://nodemailer.com/)
* [Jest](https://jestjs.io/)

## Testing
1. Create a `.env` file and fill in the variables;
2. run `yarn test`.

## Running

### Development

* run `yarn dev`

### Production

* run `yarn start`
